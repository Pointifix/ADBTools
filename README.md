# ADBTools

Android Debug Bridge Tools Editor Extension

## Features

Provides useful tools and shortcuts for working with ADB (Android Debug Bridge) directly from within VS Code.

## Requirements

You must have `adb` installed and available in your system's PATH.

## Extension Settings

This extension contributes the following settings:

* `adbtools.device`: Set the default device to connect to.
